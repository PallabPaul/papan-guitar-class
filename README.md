# Papan's Guitar Class

A colorful, single-page guitar practice website with four levels:

- Beginner
- Intermediate
- Pro
- Advance

Each level includes a suggested daily routine, notation examples (TAB + rhythm counting), and external lesson links.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- Built with Vite + React + TypeScript + Tailwind.
- External lesson links currently point to JustinGuitar course/category pages that contain video lessons.
